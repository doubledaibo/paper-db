Info:
	A small paper-database used for record paper's information, targetting convinient related work search and citation management.
	A web-app with java and mysql.

Prepare:
	1.Install maven.
	2.Install gradle.
	3.Clone git repo.

Get gradle wrapper:
	gradle wrapper --gradle-version 2.5
	You can use gradle wrapper by ./gradlew, it will use specified gradle version.

Run:
	gradle run

Eclipse:
	gradle eclipse

Build:
	gradle build
